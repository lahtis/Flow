**Author:** Tuomas Lähteenmäki

Model Explorer (Utility) is a small but important utility program (maintenance tool). 
Its purpose is to keep you up to date on which AI models you can use at any given time.


### 1. Model Explorer (Utility)
A maintenance tool to keep your suite up-to-date with the latest AI models.
* **Feature**: Lists all available `model_id` variants (e.g., `gemini-flash-latest`).
* **Feature**: Displays supported actions for each model.
